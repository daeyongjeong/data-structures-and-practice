# Week 5

## [Problem 1: Array Queue 구현](problem1)

- 문자들의 Queue를 테스트하는 프로그램 구현
- 명령어
  - `+<c>` : AddQ
  - `-` : DeleteQ
  - S : Show
  - Q : Quit

## [Problem 2: linked_list](problem2)

- 문자들의 리스트를 linked_list로 구현
- 명령어
  - `+<c>` : Insert c
  - `-<c>` : Delete c
  - `?<c>` : Search c
  - S : Show
  - Q : Quit
